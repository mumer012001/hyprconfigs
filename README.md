## My Arch Hyprland Config Files + Some Other Files

### Installation
After minimal Arch install (with grub), clone and execute -
```shell
pacman -Sy git
git clone https://github.com/mumer012001/hyprconfigs.git
cd ~/hyprconfigs/Scripts
./install.sh
```

> **Note**
> Please reboot after the install script completes and takes you to sddm login screen for the first time.
